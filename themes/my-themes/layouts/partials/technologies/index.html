Technologie

Swój stack technologiczny chciałbym oprzeć w większości na języku JavaScript ze względu na jego popularność i wszechstronność, która pozwala wykorzystać jeden język zarówno po stronie klienta, jak i po stronie serwera. W dodatku przy wykorzystaniu odpowiednich bibliotek i frameworków można w nim stworzyć nawet aplikacje desktopową która działa na wielu platformach.

Posiadam również doświadczenie w pisaniu aplikacji webowych w języku PHP.

Front-End

W swoich projektach staram się pisać semantyczny kod HTML oparty na metodyce BEM.

  • HTML 5
  • CSS 3
  • BEM Methodology
  • Sass
  • JavaScript
  • Vue.js

Back-End

Swoje doświadczenie backendowe opieram głównie na frameworkach PHP'owych Yii 2 i Laravel, lecz posiadam również doświadczenie w pisaniu skryptów w Node.js i tą technologie chciałbym w przyszłości rozwijać.

  • PHP
  • Yii 2
  • Laravel
  • Node.js

Bazy Danych

Wszystkie dotychczasowych projekty opierałem na systemie zarządzania bazą danych MySQL ze względu na jego popularność. Również staram się, by każda tworzona przeze mnie baza danych była znormalizowana.

  • MySQL

Inne

Poza wyżej wymienionymi technologiami, używam również języka TypeScript, ze względu na typowanie zmiennych i łatwiejsze debugowanie pisanego kodu. Dodatkowo do napisania jednego projektów użyłem frameworku Electron.js do utworzenia multiplatformowej aplikacji desktopowej w języku JavaScript, oraz chętnie korzystam z kontroli wersji GIT.

  • TypeScript
  • Electron
  • Hugo
  • GIT
  • Figma
